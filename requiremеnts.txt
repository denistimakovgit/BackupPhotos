Обязательно для прочтения

Зависимости:
  1. Класс YaUploader метод upload переменная disk_file_path содержит путь к созданной на Яндекс.Диск папке.
  Для корректной работы программы следует либо создать папку на Я.Диске с аналогичным именем, либо
  изменить путь в переменной.
  2. Для корректной работы программы следует создать два текстовых файла:
    ya_token.txt - для хранения токена для Яндекс.Диск
    vk_token.txt - для хранения токена для ВК
   В дальнейшем токен из этих файлов будут применятся в коде
  3. Класс YaUploader метод upload переменная dir_name содержит путь рабочей директории с файлом main.
  Программа написана в ОС Mac. В других ОС возможны ошибки
